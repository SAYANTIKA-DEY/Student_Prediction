# 📊 Student Performance Prediction using Machine Learning

## 📌 Project Overview
This project predicts student performance based on various academic and personal factors using Machine Learning techniques. The goal is to analyze the data and build a model that can estimate student scores.

---

## 📂 Dataset
- StudentPerformanceFactors.csv
- Contains information like study hours, attendance, previous scores, etc.

---

## ⚙️ Technologies Used
- Python 🐍
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🤖 Machine Learning Model
- Linear Regression / (you can update if you used another model)
- Train-Test Split
- Model Evaluation using MAE and R² Score

---

## 📊 Workflow
1. Data Loading
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Model Training
5. Prediction
6. Evaluation

---

## 📈 Results
The model predicts student performance based on input features and gives reasonable accuracy after training.

## 🚀 How to Run
1. Open Google Colab or Jupyter Notebook  
2. Upload the dataset  
3. Run all cells step by step
